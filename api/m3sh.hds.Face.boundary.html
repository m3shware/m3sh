boundary

property Face.boundary

Topological state.

A face is defined to be a boundary face if one of the incident edges is a boundary edge (an edge is a boundary edge if one of its two halfedges has this property).

Type:

bool

Note

A face only incident with boundary vertices is not classified as a boundary face.

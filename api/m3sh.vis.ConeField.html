ConeField

class m3sh.vis.ConeField(points, vectors, angle, radius, height, double, cap, resolution)[source]

Bases: Actor

Family of cones.

Visual representation of a family of (double) cones. Individual cones are defined by their apex, height, and base radius. Either the height or radius may be replaced by an angle. Specifiying all three values is considered as invalid input.

Parameters:

  • points (array_like) – Point cloud in 3-space, one point per row. Each point defines the apex of a cone.

  • vectors (array_like) – Vector field in 3-space, one vector per row. Vectors define cone axis directions and don’t have to be normalized.

  • angle (array_like) – One angle per cone. Angles are measured in radians.

  • radius (array_like) – One positive radius per cone.

  • height (array_like) – One non-negative height per cone.

  • double (bool) – True will generate double cones.

  • cap (bool) – True to close the cone along its base.

  • resolution (int) – The number of vertices along the base circle.

Note

One of angle, height, radius has to be explicitly set the None during initialization. Note that angle refers to half the opening angle of a cone.
